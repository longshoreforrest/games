# Make Game Harder

## Request
- Tee pelistä vaikeampi (Make the game harder)
- Tee niin että super alkaa 5x speed ja vaikea 2.5x speed ja normal 1x speed ja helppo o.5xspeed (Set specific start speeds)

## Changes
- Updated difficulty settings to specific user request:
    - Easy: 0.5x start speed
    - Normal: 1.0x start speed (Standard)
    - Hard: 2.5x start speed (Fast)
    - Super: 5.0x start speed (Insane)
- Adjusted max speeds and acceleration to scale with these new starting points.
- Improved collision detection to handle high speeds (swept collision check).
- Increased lane switching responsiveness for higher difficulties.
