# Add Mobile Touch Controls

## Request
Voitko muuttaa peliä niin että sitä voi pelata myös kännykällä missä ei ole näppäimistöä.

## Description
Add mobile-friendly controls to enable gameplay on touch devices without a keyboard.

## Features
- Swipe gestures: swipe left/right to change lanes
- On-screen touch buttons at bottom of screen
- Responsive design that shows appropriate controls for device type
- Mobile-optimized HUD sizing

## Changes
- `game.js`: Added touch event handlers for swipe detection and button controls
- `index.html`: Added mobile control buttons and touch hint text
- `index.css`: Added styles for mobile buttons and responsive design
